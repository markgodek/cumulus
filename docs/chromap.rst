Run Chromap to align and preprocess chromatin profiles
----------------------------------------------------------------------

This ``chromap`` workflow aligns and preprocesses FASTQ data using Chromap.

----------------------------

Workflow inputs
^^^^^^^^^^^^^^^^^^
